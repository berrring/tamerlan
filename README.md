# Tamerlan Longread

Интерактивный сайт-лонгрид про Амира Тимура (Тамерлана) и его влияние на историю региона.

## Что это за сайт

Это визуальный исторический лонгрид с прокруткой по главам:

- обложка и вступление;
- исторический контекст;
- военные походы;
- влияние на регион;
- историческая память;
- раздел источников.

Сайт сделан как storytelling-лента: при скролле меняются слайды, текстовые блоки и изображения, есть закрепленные сцены и плавные анимации.

## Технологии

- `React`
- `TypeScript`
- `Vite`
- `GSAP + ScrollTrigger`

## Запуск локально

```bash
npm install
npm run dev
```

Сборка production:

```bash
npm run build
```

## Структура

- `src/data/timurContent.ts` — основной контент (тексты, подписи, изображения).
- `src/components/Scene.tsx` — логика глав и скролл-анимаций.
- `src/components/Media.tsx` — медиа-блоки.
- `public/assets/timur/` — изображения проекта.
